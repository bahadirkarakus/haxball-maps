# HaxBall Maps

Custom HaxBall stadium files (`.hbs`) with unique mechanics.

## How to Load

1. Open HaxBall and create a room
2. Go to **Tools → Map Editor → Load from File**
3. Select the `.hbs` file

---

## Maps

### Nexus

A competitive 3-lane stadium with a hexagonal boundary and floating platforms.

```
  /‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾\
  |    [ top lane — narrow & fast ]            |
  |        [══════ PLATFORM ══════]             |
G |        [   center lane — main  ]            | G
O |        [══════ PLATFORM ══════]             | O
A |    [ bottom lane — narrow & fast ]          | A
L |                                             | L
  \___________________________________________/
```

**Features:**

- **Hexagonal boundary** — corner ramps (gold) redirect the ball inward instead of trapping it in dead corners
- **Two floating platforms** (cyan) — split the field into 3 lanes; controlling the center platform area dominates possession
- **Goal guards** (orange) — diagonal walls in front of each goal post that force angled shots; trivial straight shots from wide are blocked
- **3-lane gameplay** — top/bottom lanes are narrower (85 units) and reward precision, the center lane is wide open for power plays

**Recommended:** 2v2 to 4v4

### Rampa & Tünel Arenası

A compact arena with corner ramps, tunnel pockets at the top and bottom walls, and high-bounce deflectors (bCoef 2.5) in front of each goal.

**Recommended:** 2v2 to 3v3
